# IoT Intrusion Detection System using ML and Game Theory

## 📌 Overview
This project implements an Intrusion Detection and Prevention System (IDPS) for IoT networks using Machine Learning and Game Theory.

The system detects multiple types of cyber attacks and suggests optimal defense strategies.

---

## 🚀 Features
- Multi-class attack detection
- Random Forest classifier
- Game-theoretic defense mechanism
- High accuracy (~98%)
- Real-world dataset (CICIDS)

---

## ⚙️ Methodology
1. Data Collection from CICIDS dataset  
2. Preprocessing and feature selection  
3. Model training using Random Forest  
4. NS3-based test data generation  
5. Attack prediction  
6. Strategy selection using game theory  

---

## 🧠 Attack Types
- Normal Traffic  
- Data Mining  
- Eavesdropping  
- Tracking  
- Cyber Espionage  

---

## 📊 Results
- Accuracy: ~98%  
- Strong confusion matrix performance  
- Balanced classification across all attack types  

---

## 📁 Dataset
Dataset used: CICIDS2017  

Due to large size, it is not included in this repository.  
Download from: https://www.unb.ca/cic/datasets/ids-2017.html

## 📌 Conclusion
The system effectively detects cyber attacks and applies optimal defense strategies using a game-theoretic approach, making it suitable for real-world IoT security applications.
